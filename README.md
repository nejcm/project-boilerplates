- **react-ts-boilerplate**

## 💿 Usage

1. Clone the repo

```shell
git clone https://github.com/nejcm/project-boilerplates.git my-app && cd my-app
```

2. Pick a boilerplate

```shell
cd react-boilerplate
```

3. Install the deps

```shell
npm install
```

4. Run the dev server

```shell
npm start
```

5. Enjoy! 🎉
